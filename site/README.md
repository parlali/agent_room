# Agent Room Marketing Site

This directory contains the static Vite site for `openagentroom.com`.

Run it locally:

```bash
bun run site:dev
```

Build the static output:

```bash
bun run site:build
```

Preview the production build:

```bash
bun run site:preview
```

The site reuses the root React, Vite, Tailwind, Geist, brand mark, and typed product capability constants. It does not import authenticated routes, server functions, runtime services, credentials, local deployment state, or room data.
