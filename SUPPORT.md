# Support

Agent Room is early self-hosted software.

## Getting Help

For setup questions, bugs, and feature requests, use GitHub issues.

Before opening an issue:

- Search existing issues.
- Check `README.md`.
- Check whether Docker containers are healthy.
- Collect relevant logs without secrets.

Useful commands:

```bash
docker compose ps
docker compose logs app
```

## Do Not Share

Do not paste:

- Provider API keys
- OAuth tokens
- Generated root passwords
- `.env` files
- `.agent-room` contents
- Database dumps
- Private room files
- Private logs containing prompts, credentials, or customer data

## Security

For vulnerabilities, follow `SECURITY.md` instead of opening a public issue.
