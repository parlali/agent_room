# Code of Conduct

## Our Pledge

We want Agent Room to be a constructive, respectful project for people building self-hosted agent infrastructure.

Participants are expected to be considerate, direct, and generous with context. Disagreement is welcome when it is focused on the work and handled with care.

## Expected Behavior

- Be respectful and patient.
- Assume good intent while still naming risks clearly.
- Keep criticism specific and actionable.
- Respect privacy and security boundaries.
- Avoid sharing private credentials, private user data, or personal infrastructure details.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Publishing private information without consent
- Sharing secrets, tokens, private logs, or private room data
- Disruptive or bad-faith participation
- Security disclosure pressure outside the security policy

## Enforcement

Maintainers may remove comments, close issues, block accounts, or take other reasonable action to protect the project and its contributors.

Security issues should follow `SECURITY.md`.
